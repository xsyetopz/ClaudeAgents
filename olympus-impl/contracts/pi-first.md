# Pi-First Contract

Olympus targets Pi first.

Use Pi-native surfaces:

- `.pi/settings.json`
- `.pi/prompts/*.md`
- `.pi/skills/**/SKILL.md`
- `.pi/extensions/*.ts` or `.pi/extensions/*/index.ts`
- package manifests with `pi.extensions`, `pi.skills`, `pi.prompts`, and `pi.themes`

Use official Pi docs as source of truth when implementing syntax.
